## Context
Use this prompt when you need to structure a story that illustrates a key lesson or point for a specific audience.

## Prompt
```
Tell story about [situation] to illustrate [lesson/point]. Structure: 1) Setup, 2) Conflict/challenge, 3) Turning point, 4) Resolution, 5) Takeaway. Length: [time]. Relatable to [audience].
```
