## Context
Use this prompt for partner briefings to compile issue timelines, client concerns, actions taken, internal discussions, and resolution proposals.

## Prompt
```
Search for all communications about the issue with [Client Name] regarding [Topic]. Compile:
- Issue timeline
- Client concerns expressed
- Actions taken
- Internal discussions
- Resolution proposals
- Current status

I need this for partner briefing.
```
