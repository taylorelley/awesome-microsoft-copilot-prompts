## Context
Use this prompt when you want a concise, actionable summary of a book without reading the entire volume.

## Prompt
```
Summarize [Book Title] by [Author] for busy professional: 1) Core thesis (1 sentence), 2) Arguments (3-5 bullets), 3) Actionable insights, 4) Who should read, 5) Can I skip it?
```
