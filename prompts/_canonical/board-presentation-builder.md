## Context
Use this prompt when you need to create a board-level presentation with strategic recommendations, financial impact, and risk assessment.

## Prompt
```
Create a board-level presentation on [TOPIC] with: 1) Compelling opening slide with key message, 2) Situation analysis (current state, challenge, opportunity), 3) Strategic recommendation with supporting rationale, 4) Financial impact and business case, 5) Implementation approach and timeline, 6) Risk assessment and mitigation, 7) Decision required from board, 8) Appendix with supporting data. Use executive-appropriate visuals and limit text per slide. 10-12 slides total.
```
