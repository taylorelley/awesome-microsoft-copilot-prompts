## Context
Use this prompt when you need to create a scannable, action-oriented reference guide for a process or topic.

## Prompt
```
Create reference guide for [process/topic] for desk. Include: 1) Common use cases, 2) Step-by-step each, 3) Mistakes to avoid, 4) Troubleshooting, 5) Resources. Scannable and action-oriented.
```
