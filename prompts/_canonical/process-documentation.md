## Context
Use this prompt when you need to write step-by-step process documentation that a first-timer can follow successfully.

## Prompt
```
Document [task/workflow]. Format: 1) When to use, 2) What you need, 3) Step-by-step, 4) Common mistakes/avoid, 5) Know you're done, 6) Troubleshooting. For first-timers.
```
