## Context
Use this prompt when you need to extract tasks, due dates, dependencies, and stakeholders from an email thread into a clear actionable task list.

## Prompt
```
For the email thread below, extract:
* All tasks involving me
* Due dates
* Dependencies
* Stakeholders
Create a clear actionable task list.
```
