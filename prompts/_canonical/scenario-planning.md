## Context
Use this prompt when you need to anticipate different future possibilities and prepare strategies for best-case, most-likely, and worst-case scenarios.

## Prompt
```
Planning: [situation/project]. Create 3 scenarios: 1) Best case (optimistic/realistic), 2) Most likely (current trends), 3) Worst case (pessimistic/possible). Each: What happens, respond how, early indicators.
```
