## Context
Use this prompt when you need to deliver constructive feedback that focuses on specific observations and growth.

## Prompt
```
Give feedback to [person] about [situation/behavior]. Constructively: 1) Specific observation, 2) Impact, 3) Desired outcome, 4) Offer support. Focus on growth.
```
