## Context
Use this prompt when you need to prepare for a difficult workplace conversation such as delivering bad news or addressing performance issues.

## Prompt
```
I need to [deliver bad news/decline request/address performance] with [person/team]. Help structure: 1) Opening, 2) Main message, 3) Reasoning, 4) Next steps, 5) Positive closing
```
