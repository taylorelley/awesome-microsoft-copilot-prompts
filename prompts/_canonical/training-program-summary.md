## Context
Use this prompt to gather training program context and feedback from emails and SharePoint for program planning and improvement.

## Prompt
```
Find information about [training program name] in emails and SharePoint. Summarize:
- Program objectives
- Content covered
- Feedback received
- Scheduling or logistics mentioned

This helps me prepare for training planning discussions.
```
