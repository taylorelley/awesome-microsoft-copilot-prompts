## Context
Use this prompt when you need to structure a presentation that engages your audience and delivers a clear message within a set timeframe.

## Prompt
```
Presenting [topic] to [audience] for [duration]. Goal: [inform/persuade/inspire]. Outline: 1) Hook, 2) Why it matters to them, 3) Main sections (3-5), 4) Key data/examples, 5) Memorable closing. Constraints: [any]
```
