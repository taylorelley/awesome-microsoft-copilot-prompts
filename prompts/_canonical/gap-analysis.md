## Context
Use this prompt when you need to identify what's missing between your current state and your desired state, and create a plan to bridge that gap.

## Prompt
```
Current: [where]. Desired: [where want]. Analyze gap: 1) What's missing, 2) Root causes, 3) Bridge strategies, 4) Quick wins vs. long-term, 5) Priority order
```
