## Context
Use this prompt when you need to extract goals, KPIs, and commitments from emails and build an OKR tracking sheet.

## Prompt
```
Extract key goals, KPIs, and commitments implied or stated across all emails from this quarter. Build a proposed OKR list with: 1) Objective (what we're trying to achieve), 2) Key results (success measures), 3) Current status or progress mentioned, 4) Deadlines or target dates, 5) Stakeholders and owners, 6) Dependencies or blockers. Output as an Excel-ready OKR tracking sheet with columns for each element.
```
