## Context
Use this prompt when you need to audit a dataset for quality issues including missing values, duplicates, outliers, and inconsistencies.

## Prompt
```
Audit this dataset for quality issues: 1) Missing values (count and % by column), 2) Duplicates or inconsistencies, 3) Outliers that may be errors vs. legitimate, 4) Data type mismatches, 5) Formatting inconsistencies, 6) Referential integrity issues (if relational), 7) Recommended cleaning steps prioritized by impact. Create a data quality scorecard and cleaned dataset.
```
