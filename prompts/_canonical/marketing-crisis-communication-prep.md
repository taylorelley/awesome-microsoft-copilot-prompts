## Context
Use this prompt for crisis response planning to compile the timeline of events, stakeholders, actions taken, messaging used, and outstanding concerns.

## Prompt
```
Search for all internal discussions about [Issue/Situation]. Compile:
- Timeline of events as discussed
- Stakeholders involved
- Actions taken
- Messaging used so far
- Outstanding concerns

This is background for crisis response planning. All external communications require leadership approval.
```
