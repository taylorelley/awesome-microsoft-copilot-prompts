## Context
Use this prompt when you need to create a concise, data-driven executive brief optimized for C-suite readership.

## Prompt
```
Create an executive brief on [TOPIC] formatted as follows: 1) Executive Summary (3-4 sentences), 2) Background and Context (1 paragraph), 3) Key Findings or Analysis (3-5 bullet points), 4) Implications and Impact, 5) Recommendations (ranked by priority), 6) Next Steps with owners and deadlines, 7) Appendix (supporting data). Length: 2 pages max. Tone: professional, concise, data-driven. Optimize for C-suite readership.
```
