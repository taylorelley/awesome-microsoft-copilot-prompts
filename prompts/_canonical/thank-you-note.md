## Context
Use this prompt when you need to write a warm, genuine thank-you message to a colleague, client, or contact.

## Prompt
```
Write professional thank you to [person] for [reason]. Mention: [specific help]. Tone: genuine and warm. Length: 3-4 sentences.
```
