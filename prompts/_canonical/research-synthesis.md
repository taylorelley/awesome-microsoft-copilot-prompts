## Context
Use this prompt when you have collected notes or findings from multiple sources and need to synthesize them into a coherent overview.

## Prompt
```
Researched [topic], found: [paste notes]. Synthesize: 1) Overall themes, 2) Conflicts, 3) Knowledge gaps, 4) Next steps for deeper understanding
```
