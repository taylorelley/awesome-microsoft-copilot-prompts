## Context
Use this prompt when you need a complex topic explained in simple terms using analogies and real-world examples.

## Prompt
```
Explain [complex topic] to me like I'm [age/profession with no background]. Use analogies and real-world examples. 3-5 key concepts. No jargon unless explained.
```
