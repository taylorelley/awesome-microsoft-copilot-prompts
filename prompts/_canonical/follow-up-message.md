## Context
Use this prompt when you've sent a message and haven't received a response, and need a follow-up that adds value.

## Prompt
```
Sent [message type] to [person] about [topic] on [date], no response. Follow-up should: 1) Reference original, 2) Add new value, 3) Clear CTA, 4) Not sound desperate
```
