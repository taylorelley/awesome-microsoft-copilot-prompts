## Context
Use this prompt when you've collected feedback from multiple sources and need to distill it into actionable themes and insights.

## Prompt
```
Received feedback from [sources]: [paste]. Synthesize: 1) Common themes, 2) Contradictions, 3) Outlier opinions, 4) Actionable insights, 5) What needs clarification
```
