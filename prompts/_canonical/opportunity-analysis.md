## Context
Use this prompt when you need to assess a potential opportunity with a structured cost-benefit and ROI analysis before deciding to pursue it.

## Prompt
```
Opportunity: [describe]. Analyze: 1) Actual opportunity? 2) Requirements (time/money/skills), 3) Potential ROI (best/realistic), 4) Opportunity cost, 5) Go/No-Go with reasoning
```
