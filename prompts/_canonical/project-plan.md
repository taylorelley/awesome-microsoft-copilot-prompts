## Context
Use this prompt when you need to create a detailed project plan with milestones, deliverables, dependencies, and success metrics.

## Prompt
```
Plan [project name] with goal [outcome]. Timeline: [duration]. Team: [size/roles]. Create plan: 1) Milestones, 2) Deliverables, 3) Dependencies/risks, 4) Resources, 5) Success metrics. Format as timeline.
```
