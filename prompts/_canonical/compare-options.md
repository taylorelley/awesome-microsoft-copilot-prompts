## Context
Use this prompt when you need to compare two or more options side-by-side to make an informed decision.

## Prompt
```
Decide between [Option A] and [Option B] for [purpose]. Compare: 1) Pros/cons each, 2) Cost, 3) Time investment, 4) Best use cases, 5) Recommendation with reasoning
```
