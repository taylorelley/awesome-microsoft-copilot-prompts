## Context
Use this prompt when you need to understand how two concepts relate to each other and when they complement or conflict.

## Prompt
```
How are [Concept A] and [Concept B] related? Explain: 1) Core connection, 2) Understanding aid, 3) When together, 4) When conflicting, 5) Real-world example
```
