## Context
Use this prompt for coordination planning to summarize agreements, key contacts, deliverables, timelines, and concerns from cross-team collaboration discussions.

## Prompt
```
Search for discussions about collaboration with [Other Team/Department] on [Topic]. Summarize:
- What was agreed upon
- Who the key contacts are
- Deliverables committed
- Timeline discussed
- Open items
- Any conflicts or concerns

This is for coordination planning.
```
