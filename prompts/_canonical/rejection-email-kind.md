## Context
Use this prompt when you need to decline a request or offer while keeping the relationship positive.

## Prompt
```
Decline [request/offer] from [person] because [reason]. Message: 1) Thanks for opportunity, 2) Declines clearly, 3) Explains briefly if appropriate, 4) Keeps relationship positive
```
