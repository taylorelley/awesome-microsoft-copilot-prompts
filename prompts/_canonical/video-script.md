## Context
Use this prompt when you need to write a timed video script with a strong hook and clear structure for any platform.

## Prompt
```
[Length] video script about [topic] for [platform]. Audience: [who]. Structure: 1) Hook (3 seconds), 2) Problem/question, 3) Solution, 4) Key points (numbered), 5) CTA. Include timing cues.
```
