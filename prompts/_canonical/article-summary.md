## Context
Use this prompt when you need to quickly extract the main argument, key points, and actionable takeaways from an article.

## Prompt
```
Summarize article in 3 sections: 1) Main argument (2-3 sentences), 2) Key points (bullets), 3) Actionable takeaways. [Paste content]
```
