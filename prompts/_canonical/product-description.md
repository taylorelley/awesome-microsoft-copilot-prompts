## Context
Use this prompt when you need to write persuasive product copy that focuses on benefits rather than features for a specific customer type.

## Prompt
```
Describe [product] targeting [customer type]. Include: 1) Opening (main benefit, not feature), 2) Features as benefits, 3) Perfect for who, 4) What's different, 5) Clear CTA. Length: [words]. Tone: persuasive not salesy.
```
