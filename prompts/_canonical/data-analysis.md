## Context
Use this prompt when you have raw data or metrics and need actionable business insights without statistical jargon.

## Prompt
```
Analyze this data: [paste]. Provide: 1) Key trends/patterns, 2) Outliers/anomalies, 3) What it tells us, 4) Issues/red flags, 5) Recommended actions. Business terms, not stats jargon.
```
