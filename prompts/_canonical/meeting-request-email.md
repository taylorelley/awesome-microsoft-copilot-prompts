## Context
Use this prompt when you need to draft a meeting request that clearly communicates the topic, agenda, and importance of attendance.

## Prompt
```
Draft meeting request for [topic] with [attendees]. Details: [date/time/duration]. Agenda: [key topics]. Make clear why attendance matters.
```
