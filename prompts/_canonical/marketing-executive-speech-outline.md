## Context
Use this prompt to draft speech outlines for executives with opening hooks, main points, and closings based on topic, audience, and duration.

## Prompt
```
Draft a speech outline for [Executive Name] for [Event/Occasion]. Based on:
- Topic: [Topic]
- Audience: [Audience]
- Duration: [X minutes]
- Key messages to convey

Include opening hook, main points, and closing. I will refine with the executive.
```
