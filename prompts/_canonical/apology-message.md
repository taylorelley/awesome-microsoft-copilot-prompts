## Context
Use this prompt when you need to write a sincere apology that takes responsibility and outlines prevention steps.

## Prompt
```
Apologize for [situation] to [person/team]. Message should: 1) Take responsibility (no excuses), 2) Explain what happened, 3) Show prevention steps, 4) Request to make it right
```
