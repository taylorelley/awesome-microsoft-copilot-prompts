## Context
Use this prompt when you need to write a platform-specific social media caption that grabs attention and drives engagement.

## Prompt
```
Write [LinkedIn/Instagram/Twitter] caption for [content type] about [topic]. Tone: [professional/casual/humorous]. Include: 1) Hook first line, 2) Value in middle, 3) CTA end. Best practices: [hashtags/length/emoji]
```
