## Context
Use this prompt to compile event planning information including logistics, speakers, promotions, budget, attendees, and action items from discussions.

## Prompt
```
Search for discussions about [Event Name]. Compile all information about:
- Logistics and venue discussions
- Speaker or content planning
- Promotional activities mentioned
- Budget discussions
- Attendee targets
- Action items and owners

Format as event planning summary.
```
