## Context
Use this prompt when you need to understand the background, key actors, and different perspectives behind a current event.

## Prompt
```
Understand [current event]. Give context: 1) How we got here (key events), 2) Main actors/motivations, 3) Different perspectives, 4) What to read/watch
```
