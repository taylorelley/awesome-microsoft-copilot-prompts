## Context
Use this prompt when you are planning an action or project and need to proactively identify, evaluate, and mitigate potential risks.

## Prompt
```
Planning: [action/project]. Assess risks: 1) What could go wrong (brainstorm), 2) Likelihood each (high/medium/low), 3) Impact (high/medium/low), 4) Top 3 to mitigate, 5) Strategies each, 6) Early warning signs
```
