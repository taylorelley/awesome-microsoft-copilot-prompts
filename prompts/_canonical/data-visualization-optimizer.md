## Context
Use this prompt when you need to transform a data-heavy presentation for maximum impact with better visuals, hierarchy, and clarity.

## Prompt
```
Transform this data-heavy presentation for maximum impact: 1) Replace text with visuals where possible, 2) Suggest best chart types for each dataset, 3) Add visual hierarchy and emphasis, 4) Create 'so what' statements for each insight, 5) Simplify complex slides (one message per slide), 6) Add executive summary slides, 7) Ensure color consistency and brand alignment. Prioritize clarity and memorability.
```
