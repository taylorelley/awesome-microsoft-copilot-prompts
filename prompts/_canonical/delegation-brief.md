## Context
Use this prompt when you need to delegate a task and want to provide clear context, success criteria, and support resources.

## Prompt
```
Delegate [task] to [person/role]. Create brief: 1) What needs doing (specific outcome), 2) Why it matters, 3) Success criteria, 4) Resources available, 5) Deadline/check-ins, 6) Who to ask if stuck
```
