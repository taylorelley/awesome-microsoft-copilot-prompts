## Context
Use this prompt when you need to optimize a sales presentation with a stronger hook, value proposition, social proof, and objection handling.

## Prompt
```
Optimize this sales presentation for winning: 1) Stronger opening hook (problem/opportunity), 2) Clear value proposition (quantified benefits), 3) Social proof and success stories, 4) Competitive differentiation slide, 5) Visual product/solution demonstration, 6) Pricing slide with value anchoring, 7) Objection handling (anticipated concerns), 8) Strong call-to-action and next steps. Ensure flow builds to decision. 15-20 slides.
```
