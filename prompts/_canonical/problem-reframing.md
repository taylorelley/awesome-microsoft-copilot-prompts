## Context
Use this prompt when you feel stuck on a problem and need alternative perspectives to unlock new solutions.

## Prompt
```
Problem: [current thinking]. Reframe: 1) Real problem or symptom? 2) Who affected (stakeholders)? 3) Solving wrong problem? 4) Alternative framings, 5) Which opens most solutions?
```
