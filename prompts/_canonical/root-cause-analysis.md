## Context
Use this prompt when you need to dig beneath surface symptoms to identify and address the true root cause of a problem.

## Prompt
```
Problem: [describe]. Find root cause: 1) Surface symptoms, 2) Ask 'why' 5 times, 3) Potential root causes ranked, 4) How to verify actual cause, 5) Solutions addressing root, not symptoms
```
