## Context
Use this prompt when you need to break down a goal into actionable milestones with identified obstacles and progress metrics.

## Prompt
```
Achieve [goal] by [timeframe]. Current: [where now]. Create plan: 1) Milestones, 2) Obstacles/mitigation, 3) Resources needed, 4) Progress measurement, 5) Accountability. Realistic given [constraints].
```
