## Context
Use this prompt to generate blog post outlines with headline options, introduction hooks, main sections, CTAs, and SEO considerations for review and expansion.

## Prompt
```
Based on the topic [Topic] and our discussions about [Product/Service], draft a blog post outline including:
- Suggested headline options (3)
- Introduction hook
- Main sections with key points
- Call-to-action suggestions
- SEO considerations for [Target Keyword]

This is a starting outline for my review and expansion.
```
