## Context
Use this prompt when you need to outline a blog post that captures attention, delivers value, and drives action from your target audience.

## Prompt
```
Blog outline about [topic] for [audience]. Goal: [inform/persuade/entertain]. Structure: 1) Attention-grabber, 2) 3-5 main sections, 3) Examples/data, 4) Strong CTA conclusion. Target: [words]
```
