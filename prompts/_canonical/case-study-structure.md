## Context
Use this prompt when you need to structure a case study that showcases a project's challenges, approach, and measurable results.

## Prompt
```
Case study for [project/client/situation]. Include: 1) The Challenge, 2) The Approach, 3) Results (quantified), 4) Key Learnings, 5) Client testimonial (draft request). Target: [words]
```
