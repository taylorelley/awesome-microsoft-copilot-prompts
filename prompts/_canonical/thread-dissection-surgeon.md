## Context
Use this prompt when you need to untangle a complex, messy email thread and get a clear resolution plan with draft messaging.

## Prompt
```
Dissect this messy email thread into: 1) What each person said (chronological summary), 2) Hidden assumptions each person is making, 3) Conflicts or disagreements (overt and subtle), 4) Stalled actions and why they're stuck, 5) Unanswered questions that need resolution, 6) Decisions that need to be made, 7) Recommended resolution plan, 8) Draft messaging to move the thread forward. Keep summary concise and actionable.
```
