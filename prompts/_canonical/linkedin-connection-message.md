## Context
Use this prompt when you want to send a personalized LinkedIn connection request that is professional and concise.

## Prompt
```
Write LinkedIn connection to [person's role] at [company]. Context: [how you found them/mutual connection/why]. Personalized and professional. 2-3 sentences max.
```
