## Context
Use this prompt when you have existing content and want to adapt it into multiple formats to reach a wider audience.

## Prompt
```
Have this content: [describe/paste]. Repurpose into: 1) [Format 1], 2) [Format 2], 3) [Format 3]. For each: key points to emphasize and format guidelines.
```
