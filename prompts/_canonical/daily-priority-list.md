## Context
Use this prompt when you have a list of tasks and need to prioritize them by urgency and importance within your available time.

## Prompt
```
These tasks today: [list]. Prioritize: 1) Rank urgency x importance, 2) Group related, 3) Estimate time each, 4) Identify delegatable/deletable, 5) Create realistic schedule ([hours] available). Flag urgent but not important.
```
