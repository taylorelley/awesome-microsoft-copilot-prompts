## Context
Use this prompt when you want to learn a new skill and need a structured roadmap tailored to your current level and available time.

## Prompt
```
Learn [skill] for [purpose]. Available: [time per week]. Current: [level]. Roadmap: 1) Skills in order, 2) Resources (free preferred), 3) Exercises, 4) Progress indicators
```
