## Context
Use this prompt when you need to create an engaging training presentation with learning objectives, interactive elements, and knowledge checks.

## Prompt
```
Create an engaging training presentation on [TOPIC] for [AUDIENCE]. Include: 1) Learning objectives (what attendees will know/do), 2) Key concepts with visual explanations, 3) Real-world examples and scenarios, 4) Interactive elements (polls, questions, exercises), 5) Do's and don'ts or best practices, 6) Knowledge check slides, 7) Resources and next steps, 8) Appendix with FAQs. Use adult learning principles and keep it interactive. 20-30 slides.
```
