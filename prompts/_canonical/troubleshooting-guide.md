## Context
Use this prompt when you need to build a step-by-step troubleshooting guide for a recurring problem or technical issue.

## Prompt
```
Problem: [what's broken]. Troubleshooting guide: 1) Symptoms checklist, 2) Most common causes (ranked), 3) Diagnostic steps, 4) Solutions per cause, 5) Escalate vs. fix, 6) Prevention
```
