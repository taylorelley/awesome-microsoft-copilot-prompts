## Context
Use this prompt before technical review meetings to gather technical progress, challenges, design decisions, open questions, resource needs, and timeline concerns.

## Prompt
```
I have a technical review meeting for [Project Name]. Search for:
- Technical progress discussed
- Challenges raised by the team
- Design decisions made
- Open questions or debates
- Resource needs mentioned
- Timeline concerns

Compile as technical review prep notes.
```
