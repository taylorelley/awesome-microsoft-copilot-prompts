## Context
Use this prompt to compile facilities issues, resolution status, recurring problems, vendor involvement, cost implications, and outstanding items from discussions.

## Prompt
```
Search for discussions about facilities issues over the past month. Compile:
- Issues reported
- Resolution status
- Recurring problems
- Vendor involvement
- Cost implications mentioned
- Outstanding items

Format as facilities status report.
```
