## Context
Use this prompt when you face a complex decision with multiple options and need a structured evaluation with scoring and second-order effects.

## Prompt
```
Decide: [decision]. Options: [list]. Evaluate: 1) Criteria for good decision, 2) Score options, 3) Second-order effects, 4) Reversibility, 5) Recommendation with reasoning
```
